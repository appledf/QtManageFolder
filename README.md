## 文件夹管理工具

### 工具描述：
以文件夹名称按照固定格式进行命名来管理

### 功能：

- 选定路径
- 选定导出路径
- 当前数据表格内容导出成csv文件
- 创建新文件夹
- 修改文件夹名称
- 表格内容过滤，排序
- 系统参数，文件夹名称解析规则
